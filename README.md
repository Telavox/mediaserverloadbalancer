# Service Template

Here you can find any template we support for implementing new service. To see different templates, check the branches available in this template repository!

## Why We Need This
The goal is to keep any template service in this repository regardless of the technology. The main goal of a template repository is to give you the exact folder structure when you create your repository from it. Any information about the technology should be written in the documentation. Each branch (template) should have its own documentation. Branches in template repositories are not relevant to each other and can not be merged. Also, the history of each will be independent of each other. You can create a new template by pushing your template to a new branch.

To learn more about template repositories, you can read this [documentation](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-template-repository)

## How To Clone

- Select the template you want to clone from the branches drop down
- Follow this [documentation](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-repository-from-a-template) to create a repository from it
